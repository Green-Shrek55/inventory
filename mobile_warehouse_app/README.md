# Mobile Warehouse App

Flutter-клиент для роли складовщика. Приложение не хранит отдельную базу данных:
все данные синхронизируются с текущим Spring Boot сервером через `/api/warehouse/**`.

## Запуск

1. Запустите Spring Boot сайт, например на `http://10.124.145.191:9092`.
2. В приложении укажите адрес сервера.
3. Войдите логином складовщика или администратора. В демо-проекте пароль для ролей: `admin123`.
4. Выберите корпус и работайте с приемкой, размещением, инвентаризацией и утилизацией.

## Android

Для сборки требуется установленный Flutter SDK:

```powershell
cd mobile_warehouse_app
flutter pub get
flutter run
```
