Резервное копирование

Выход

По кнопке ниже выгружается ZIP-архив со всеми ключевыми таблицами (пользователи, справочники, оборудование, заявки, логи). Формат JSON — подходит для восстановления и переноса в PostgreSQL.

Скачать backup.zip

Советы по безопасности

  • Храните резервные копии на защищённом носителе.
  • Делайте выгрузку как минимум раз в неделю.
  • После восстановления проверяйте журнал действий.
